## 四、技术实现方案（简化版）

### 4.1 数据存储

**方案**: 使用localStorage + IndexedDB

**存储内容**:
```typescript
interface UserData {
  learningProgress: Map<string, Progress>; // 学习进度
  favorites: string[]; // 收藏列表
  wrongQuestions: string[]; // 错题列表
  theme: 'dark' | 'light'; // 主题偏好
  lastVisit: string; // 最后访问时间
}
```

**优势**:
- 无需后端
- 实现简单
- 用户体验好

---

### 4.2 状态管理

**方案**: 使用Zustand（轻量级）

**核心Store**:
```typescript
// 用户数据Store
const useUserStore = create(
  persist(
    (set) => ({
      progress: {},
      favorites: [],
      wrongQuestions: [],
      
      updateProgress: (questionId, status) => 
        set((state) => ({
          progress: { ...state.progress, [questionId]: status }
        })),
      
      toggleFavorite: (questionId) => 
        set((state) => ({
          favorites: state.favorites.includes(questionId)
            ? state.favorites.filter(id => id !== questionId)
            : [...state.favorites, questionId]
        })),
    }),
    { name: 'user-data' }
  )
);
```

---

### 4.3 路由规划

**方案**: 使用React Router v6

**页面结构**:
```
/                    # 首页（题目列表）
/questions/:id      # 题目详情
/favorites          # 收藏夹
/progress           # 学习进度
```
